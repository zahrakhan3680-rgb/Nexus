# Nexus Frontend Internship Enhancements

This repository contains advanced frontend enhancements for the Nexus Investor & Entrepreneur Collaboration Platform.

## What Was Added

- Week 1: Scheduling calendar and availability/request flow
- Week 2: Video calling mock UI and Document Chamber
- Week 3: Payments simulation, security enhancements, and demo walkthrough

## Documentation

- Architecture and component structure: `docs/ARCHITECTURE.md`
- Submission tracking checklist: `docs/SUBMISSION_CHECKLIST.md`

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
